# Moi! We are n0pd!

![n0pd Banner](./bland/bannar.jpg)

n0pdは、[@kqnade](https://github.com/kqnde)が主導する技術団体およびコミュニティです。\
私たちは、主にオープンソース貢献、ソフトウェア開発、技術交流イベントの開催を通じて、技術コミュニティの発展に寄与しています。

## We are currently working on

- [n0pd/mpp](): SteamVR用フリックキーボード 
  - SteamVR環境での高速な文字入力を可能にするキーボードアプリケーション。
  - 現在のスマートフォンの多くで採用されているフリック入力方式をVR環境に最適化。
  - 技術スタック: Unity, C#, SteamVR SDK, OpenVR, Mozc
- [n0pd/lensy](): VRChat Photo Manager - VRChatで撮影した写真を自動収集・管理するデスクトップアプリケーション
  - VRChat内で撮影した写真を自動的に収集し、整理・管理するためのツール。
  - ユーザーフレンドリーなインターフェースを提供し、写真の閲覧や共有を容易に。
  - 技術スタック: Rust, Tauri, SQLite, React, TypeScript

## Our Mission

- 技術コミュニティの活性化と発展を支援すること。
- オープンソースソフトウェアの普及と貢献を促進すること。
- 技術者同士の交流と知識共有の場を提供すること。

## Our Values

私たちは **OICI(おいしい)** の原則に基づいて活動しています。
- Openess: すべての活動は透明性を持ち、オープンに行います。
- Inclusivity: 多様なバックグラウンドを持つ人々を歓迎します。
- Collaboration: 技術者同士の協力と共同作業を重視します。
- Innovation: 新しい技術やアイデアの探求を奨励します。
